validateInstrumentData¶

class primitives_GMOS.GMOSPrimitives[source]¶

This is the class of all primitives for the GMOS level of the type hierarchy tree. It inherits all the primitives to the level above , ‘GEMINIPrimitives’.

validateInstrumentData(rc)[source]¶

This primitive is called by validateData to validate the GMOS instrument specific data checks for all input files.

Parameters:logLevel (integer from 0-6, 0=nothing to screen, 6=everything to screen. OR the message level as a string (ie. ‘critical’ , ‘status’, ‘fullinfo’...)) – Verbosity setting for log messages to the screen.
